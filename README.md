# AI Reel Studio Landing Page

AI-powered Instagram Reels creation service for Indian brands.

## Tech Stack

- Next.js 16.1.6
- React 19.2.3
- TypeScript 5
- Tailwind CSS 4
- Node.js >= 20.9.0

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Features

- Single-page landing website
- Mobile-first responsive design
- 10 sections: Hero, How It Works, What You Get, AI Influencer Models, Why AI Videos, Pricing, Sample Video, FAQ, Final CTA, Footer
- Conversion-optimized layout
- Tailwind CSS 4 styling
- TypeScript components

## Project Structure
